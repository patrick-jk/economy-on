# Economy On

## This is an app made to help people controlling your finances, separating by:

- Leisure
- Personal Development
- Social
- Physiology
- Security

## App features:

- Chart (Spent money by type)
- Finances List

## Technologies

- Clean Architecture
- MVVM
- Kotlin Coroutines
- LiveData
- Retrofit
- Fragments
- Navigation Component
- SplashScreen API
- ViewBinding
